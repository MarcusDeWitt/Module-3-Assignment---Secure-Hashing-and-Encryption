# Module-3-Assignment---Secure-Hashing-and-Encryption

## Caeser Cipher
The Caesar Cipher app demonstrates a simple way of encrypting a message by taking input from the user and shifting it a certain amount of values, also determined by the user, by its ASCII code. Then it displays its encrypted or decrypted output. The Caeser Cipher Proof file showcases this.

## Hash Generator
The hash generator app demonstrates the process of hashing a users input text or file to make it more secure using SHA-256. This allows data to be kept more secure and keep its integrity, the file Hash Proof demonstrates its capabilities. 

## SSL Demo
I used SSL to demonstrate a digital signature, which is used to verify that someone is who they say they are and the message sent was by them, or to verify that the person sending it to you is who they say they are. It is also used to ensure data integrity as it can verify if a file has been tampered with. THe SSL DEMO file holds all needed demonstrations on this.
